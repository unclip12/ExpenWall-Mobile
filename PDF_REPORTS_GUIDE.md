## ğŸ“± How to Use

### Step 1: Open Report Builder

1. Tap menu icon (â˜°) in top-right
2. Select "Generate Report" (or "Reports")
3. You'll see the Report Builder screen

### Step 2: Select Template

1. Three template cards are shown
2. Tap on your preferred template:
   - **Simple Summary** - Quick overview
   - **Detailed Professional** - Comprehensive list
   - **Budget Analysis** - Budget performance
3. Selected template shows checkmark âœ“

### Step 3: Choose Date Range

**For Preset Ranges:**
1. Tap a date chip (This Month, Last Month, etc.)
2. Selected chip highlights in purple

**For Custom Range:**
1. Tap "Custom" chip
2. Two date picker buttons appear
3. Tap "Start Date" and select date
4. Tap "End Date" and select date

### Step 4: Apply Filters (Optional)

**Category Filter:**
1. Tap "Filter by Categories"
2. Check boxes for categories to include
3. Tap "Done"

**Merchant Filter:**
1. Tap "Filter by Merchants"
2. Select merchants (feature coming soon)
3. Tap "Done"

### Step 5: Configure Options

**Include Receipt Images:**
- Toggle ON to embed receipt photos in PDF
- Toggle OFF for smaller PDF size

**Include Charts:**
- Toggle ON for visual charts (pie, bar graphs)
- Toggle OFF for text-only report

**Company/Personal Name:**
1. Tap to edit
2. Enter your name or company name
3. Appears in report header
4. Leave blank for default "ExpenWall"

### Step 6: Generate PDF

1. Review all settings
2. Tap "Generate PDF Report" button
3. Wait 2-5 seconds (loading indicator shows)
4. PDF Preview screen opens automatically

### Step 7: Preview & Share

**In Preview Screen:**

1. **Zoom** - Pinch to zoom in/out
2. **Scroll** - Swipe to view all pages
3. **Share** - Tap share icon (top-right)
4. **Print** - Tap printer icon (top-right)
5. **Done** - Tap "Done" when finished

**Sharing Options:**
- System share sheet opens
- Choose WhatsApp, Email, Drive, Messages, etc.
- PDF is attached automatically

## ğŸ› ï¸ Technical Details

### File Storage

- **Location:** App documents/reports/
- **Format:** PDF (A4 size)
- **Naming:** report_[timestamp].pdf
- **Size:** 50-500 KB (depending on content)

### Report Generation

- **Engine:** pdf package (Flutter)
- **Rendering:** Vector graphics (scalable)
- **Speed:** 2-5 seconds typical
- **Background:** Safe to navigate away during generation

### Data Sources

- **Transactions:** LocalStorageService
- **Budgets:** Budget model data
- **Categories:** Transaction categorization
- **Analytics:** Calculated on-the-fly

### Privacy & Security

- **Local Storage:** PDFs stored only on device
- **No Cloud Upload:** Reports not synced automatically
- **User Control:** You decide when/where to share
- **Deletion:** Delete reports anytime from History

## ğŸ”§ Troubleshooting

### "Please select start and end dates"

**Problem:** Custom date range selected but dates not picked

**Solution:**
1. Tap "Start Date" button
2. Select a date
3. Tap "End Date" button
4. Select a date
5. Try generating again

### "Error generating report"

**Problem:** PDF generation failed

**Solutions:**
1. Check you have transactions in selected date range
2. Try a different date range
3. Restart the app
4. Clear filters and try again

### Empty/Blank Report

**Problem:** Report has no data

**Possible Causes:**
1. No transactions in selected date range
2. All transactions filtered out
3. Check category/merchant filters

**Solution:**
1. Expand date range
2. Clear filters ("All categories")
3. Verify you have transactions for that period

### "No reports yet" in History

**Problem:** Report History is empty

**Solution:**
- Generate your first report!
- Reports appear here after generation
- Check Reports list refreshes automatically

### Share Button Not Working

**Problem:** Can't share PDF

**Solutions:**
1. Grant storage permissions to app
2. Check phone has sharing apps installed
3. Try "Save to Drive" or "Email" specifically
4. Restart app and try again
